# Blackwargraymon
CCTV내 군중 밀집도 계산 및 경고 시스템

# 개요
우리가 주목한 사회적 문제: 혼잡한 거리 상황에서 발생할 수 있는 안전사고를 주제로 정하였습니다.
저희는 오픈소스 AI를 활용하여 면적 대비 혼잡도를 계산하여 일정 수준의 기준치(Threshold)를 넘으면 경고메시지를 보내는 것을 목표로 하였습니다.
